#  Deployment Guide

## Quick Start

1. **Clone and Setup:**
```bash
git clone https://github.com/adityakarthikrTR/AI-prod-gains-.git
cd AI-prod-gains-
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2. **Run Application:**
```bash
streamlit run app.py --server.port 8505
```

3. **Access URLs:**
- Local: http://localhost:8505
- Network: http://[your-ip]:8505
